# docs/

Reference material, specs, and planning artifacts live here. Add new language notes, design docs, and roadmap updates alongside `plan.md` so contributors can discover documentation in one place.

- `tokens.md` — summarizes the Runic lexer/token vocabulary and how each
  category maps to commands, pipelines, declarations, and interop features.
- `plan.md` — the current top-level project roadmap.
- `lsp.md` — the current status and roadmap for the language server.
- `migrating-from-bash.md` — step-by-step guidance for porting existing bash
  scripts to Runic using the currently supported native feature set.
